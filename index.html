What is PHP Test Fixture?

PHP Test Fixture is a simple, minimalistic and efficient PHP testing framework authored and maintained by none other than DonaldKellett. It is simple in the sense that PHP Test Fixture is very simple to operate - it does not require the command line like a few of the more popular PHP testing frameworks do
